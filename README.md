## :octocat: Setup

### Installation

```sh
# Clone this repo
git clone git@github.com:filipe1309/ew-hacker-chat-server.git

# Enter project folder
cd ew-hacker-chat-server

# Install dependencies
npm ci
```

### 🏃 Runnning

```sh
npm start
# app will run at: http://localhost:9898
```
